🔄 FLUX DE DONNÉES
──────────────────
1. User Auth
   └── useAuth() → session

2. Navigation
   └── useWeekNavigation() → selectedDate, referenceDate

3. Data Loading
   ├── usePlannerData(session, referenceDate) → days[]
   └── useRecipes(session) → recipes[]

4. UI Rendering
   ├── viewMode === "focus"
   │   ├── WeekProgressCard
   │   ├── DayGridSelector
   │   └── FocusView → DayMealCard (x2)
   │
   └── viewMode === "list"
       └── ListView → DayMealCard (x14)

5. Modals
   ├── WeekPickerModal (calendrier)
   └── RecipePickerModal (bibliothèque)

6. Notifications
   └── useToast() → Toast

📝 POUR AJOUTER UNE FEATURE
───────────────────────────
1. Hook nécessaire ? → hooks/useMyFeature.ts
2. Composant UI ? → components/MyFeatureComponent.tsx
3. Types ? → utils/types.ts
4. Intégrer → planner.tsx

Exemple : Ajouter "Shopping List"
  ├── hooks/useShoppingList.ts (logique)
  ├── components/ShoppingListModal.tsx (UI)
  ├── utils/types.ts (type ShoppingItem)
  └── planner.tsx (import et utilisation)
